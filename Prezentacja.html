Wprowadzenie

CSS (Cascading Style Sheets) jest językiem stylów używanym do opisywania wyglądu dokumentu HTML. CSS umożliwia projektantom tworzenie atrakcyjnych stron internetowych z różnymi efektami i stylami.

Selektory

Selektory są używane do określenia elementów, do których ma być zastosowany styl. Na przykład, selektor "p" odnosi się do wszystkich paragrafów w dokumencie HTML.

Właściwości CSS

Właściwości CSS są używane do określenia stylu elementów HTML. Na przykład, właściwość "color" określa kolor tekstu, a właściwość "background-color" określa kolor tła elementu.
